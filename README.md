# Chap 01 : 디자인 패턴 소개와 전략 패턴, 디자인 패턴 세계로 떠나기

## 상속을 이해하자
- 상속을 잘못사용할 시 발생하는 문제점
   - 서브클래스에서 코드가 중복된다.
   - 실행 시에 특징을 바꾸기 어렵다.
   - 모든 서브클래스의 행동(구현)을 알기 어렵다.
   - 코드를 변경했을 때 다른 서브클래스들에게 원치 않은 영향을 끼칠 수 있다.
